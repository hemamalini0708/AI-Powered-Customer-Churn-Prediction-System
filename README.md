# AI-Powered-Customer-Churn-Prediction-System
End-to-end AI-powered Customer Churn Prediction system that predicts whether a telecom customer will leave or stay based on service usage, billing details, and demographic features. Includes preprocessing, SMOTE balancing, feature selection, and comparison of multiple ML models. Final model: Gradient Boosting (AUC ~0.88). Deployed using Streamlit.
